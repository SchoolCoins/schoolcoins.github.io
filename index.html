¿Qué es SchoolCoins?

Plataforma educativa con gamificación para alumnos de primaria y secundaria. Permite a los profesores asignar tareas con códigos QR y recibir retroalimentación automática con IA.

Beneficios

  • Para escuelas: Gestión moderna y reportes automáticos.
  • Para profesores: Más tiempo para enseñar.
  • Para alumnos: Feedback inmediato y posibilidad de ganar recompensas.
